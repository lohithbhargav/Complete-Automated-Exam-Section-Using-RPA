To Run the Solution file(.atmx) there are the requirements to be there in System.
1. Automation Anywhere Enterprise Edition (Software)
2. Credentials of Enterprise Edition
3. Good Internet is must to run Bot
4. The Path for the folders have to particular.


Note: As the RPA Bot runs in the predefined rules and predefined System Settings.

Example: The Bot was trained in a resolution of 1080*1920. So, the mouse movements, mouse clicks and Path directory are w.r.t that System.
If the Testing System also has, all the same settings, Then only the Bot can run without error.
